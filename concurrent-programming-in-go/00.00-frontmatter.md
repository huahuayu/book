# Go 并发编程：从入门到实战

## 关于本书

本书旨在帮助有一定 Go 语言基础的开发者，系统性地掌握 Go 并发编程的核心概念和最佳实践。通过大量实际代码示例，你将学习如何编写高效、安全的并发程序。

## 本书特点

- **由浅入深**：从基础概念到高级模式，循序渐进
- **代码驱动**：每个知识点都配有可运行的示例代码
- **实战导向**：包含真实场景的综合案例
- **最佳实践**：涵盖常见错误和避免方法

## 读者对象

本书适合：
- 有 Go 语言基础，想深入学习并发编程的开发者
- 有其他语言并发编程经验，想学习 Go 特色并发模型的工程师
- 希望提升系统设计能力的后端开发者

## 前置知识

阅读本书前，你应该：
- 熟悉 Go 语言基本语法
- 了解函数、结构体、接口等概念
- 能够使用 `go run` 和 `go build` 运行 Go 程序
